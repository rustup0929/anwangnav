4. Hidden Answers：自由提问和答题

Hidden Answers 有点像 Reddit 和 Quora，但它完全未经审查，不受监管。它是开放的论坛，大家可以在这里提问，也可以回答其他人的问题。因为它是完全匿名的，人们可以自由地询问和回答，讨论一些平时可能害怕分享的内容。
